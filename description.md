# Analisis Viralitas Instagram Reels

## Problem Background
Instagram Reels menjadi salah satu fitur media sosial yang banyak digunakan untuk distribusi konten digital. Namun, performa konten sering dianggap tidak konsisten karena terdapat perbedaan besar pada jumlah views dan engagement meskipun jenis kontennya serupa.

Project ini bertujuan untuk menganalisis apakah algoritma Instagram benar-benar acak atau terdapat faktor tertentu yang memengaruhi viralitas sebuah Reels. Analisis dilakukan menggunakan data engagement seperti likes, comments, shares, saves, impressions, dan sumber traffic konten.

---

## Project Output
Output dari project ini adalah:
- Exploratory Data Analysis (EDA)
- Visualisasi data Instagram Reels
- Correlation analysis
- Statistical inference menggunakan ANOVA
- Insight mengenai faktor-faktor yang memengaruhi impressions dan engagement Instagram Reels

---

## Data
Dataset yang digunakan berisi data performa Instagram Reels dengan beberapa variabel seperti:
- Impressions
- Likes
- Comments
- Shares
- Saves
- From Home
- From Explore
- Follows

Dataset digunakan untuk menganalisis hubungan antar engagement dan performa konten Instagram Reels.

---

## Method
Metode yang digunakan dalam project ini meliputi:
- Data Cleaning
- Statistik Deskriptif
- Visualisasi Data
- Correlation Analysis
- ANOVA (Analysis of Variance)

ANOVA digunakan untuk mengetahui apakah terdapat perbedaan signifikan engagement berdasarkan kategori impressions.
